Mensagens

As mensagens privadas (PV) não são necessariamente um chat / bate-papo, é mais uma espécie de mural de recados.

Recomendo que você use para deixar recados importantes e/ou trocar informações e contato, pois há limitações como:

  • Turno de mensagens: quando você envia uma mensagem para um jogador, você só poderá enviar outra mensagem após ele responder e vice-versa ou após três dias, quando há o arquivamento automático das mensagens.
  • Arquivamento: a cada três dias o histórico de mensagens é arquivado caso não haja mais interação de conversa.
