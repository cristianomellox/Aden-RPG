    Guia de Mineração

    Aqui você encontrará todas as informações para dominar o sistema de mineração.

    Capturando uma Mina: Combate contra Monstros

    Esta é a forma principal de obter uma mina que não tem dono.

    • Iniciando o Combate: Para começar a disputar uma mina, você precisa iniciar um combate contra o monstro que a protege. Um jogador não pode iniciar um combate se já for dono de uma mina. O status da mina muda de "aberta" para "disputando" quando o combate começa.
    • Atacando o Monstro: O ataque é um processo contínuo, onde o dano que você causa é registrado. O sistema mantém um ranking de dano para cada mina em disputa, crucial para determinar o novo dono.
    • Finalizando a Sessão: Quando a vida do monstro chega a zero ou quando o tempo termina, o sistema verifica o ranking de dano para encontrar um novo dono. O jogador com o maior dano elegível se torna o novo proprietário. Após a captura, o ranking é zerado e a mina muda para "ocupada". Se ninguém for elegível, a mina é resetada.

    Capturando uma Mina: Combate PvP

    Depois que uma mina é capturada por um jogador, ela pode ser disputada através de um combate PvP.

    • Desafiando o Dono: Qualquer jogador pode desafiar o dono de uma mina. Você não pode desafiar a si mesmo. O combate PvP consome uma tentativa, que pode ser comprada com ouro. Essas tentativas também são resetadas todos os dias.
    • Batalha: O dono da mina ataca primeiro, seguido pelo desafiante. A batalha continua até que um dos jogadores chegue a zero de vida.
    • Consequências da Batalha: Se o desafiante vencer, ele se torna o novo dono. Se o dono atual defender, ele mantém a mina. O histórico de ataques à mina fica disponível para o dono ver.

    Sessões e Recompensas

    • Sessão: O sistema de mineração possui sessões que duram 2 horas. Um cronômetro na interface mostra quando a próxima sessão começa (em toda hora par).
    • Recompensas: Ao conquistar uma mina, você passa a coletar cristais passivamente. O máximo de cristais que você pode coletar é de 1500, equivalente ao tempo da sessão. Se um jogador capturar sua mina, você receberá os cristais coletados até aquele momento.

    Guilda Dominante

    A guilda que tiver mais membros como donos de mina, aparecerá no topo como Guilda Dominante.
